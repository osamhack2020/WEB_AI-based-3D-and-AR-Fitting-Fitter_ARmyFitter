## 팀소개
* ARmyFitter란 팀명을 사용하였으며
* 팀원은 4명(개발관리장교, 빅데이터기술담당, VR기술/개발담당, 정보작전SW개발장교)으로 구성하였음.
* 팀원 별 기획 및 계획, 시스템 설계 및 분석, 구현 및 프로젝트 개발 관리, 영상제작 등의 임무 분장을 통해 과업을 수행 함

### front-end

 - 기타 사용한 라이브러리
